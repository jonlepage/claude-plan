# Changelog

## [0.1.0] - 2026-03-25

### Added
- Inline directive annotations in markdown preview mode
- Floating "Add directive" button on text selection
- Gutter pencil icon for quick whole-line directives on hover
- Textarea input for multi-line directives
- Numbered badges (1. 2. 3.) with edit and delete controls
- One-click send to Claude Code via ✨ button or `Alt+Enter`
- Auto-paste into Claude Code chat input (Windows, via PowerShell SendKeys)
- Clipboard fallback for non-Windows platforms
- Navigation between directives with ▲▼ buttons
- Toolbar with directive count, copy, and clear controls
- Green flash confirmation on directive add
- Session persistence across tab switches
- Smart targeting for tables, lists, and nested structures
- `Insert` keyboard shortcut to add directive
- Source editor support with decorations and context menu
